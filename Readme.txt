Git Commands : 

1. git config - This command sets the author name and email address respectively to be used with your commits.

2. git init- This command is used to obtain a repository from an existing URL.

3. git add - This command records or snapshots the file permanently in the version history.

4. git diff - This command shows the file differences which are not yet staged.

5. git reset - This command undoes all the commits after the specified commit and preserves the changes locally.

6. git status - This command lists all the files that have to be committed.

7. git log - This command is used to list the version history for the current branch.

8. git rm - This command deletes the file from your working directory and stages the deletion.

9. git show - This command shows the metadata and content changes of the specified commit.

10. git tag - This command is used to give tags to the specified commit.
